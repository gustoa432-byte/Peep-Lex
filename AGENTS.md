# Project UI Rules

- **СТРОГО ЗАПРЕЩЕНО** использовать когда-либо в UI размытое стекло (классы `backdrop-blur-*`, `bg-white/10` и любые другие эффекты glassmorphism). 
- Для прозрачности используй только фон черного цвета с 5% прозрачностью (например, `bg-black/95`).
- Мы делаем все для мобильного телефона, никакого мусора для десктопа! Эффекты размытия сильно тормозят рендер на мобильных устройствах.
